# Itsycal Software — Portable Menu Bar Calendar

## About Itsycal Software

**Itsycal Software** provides a complete minimal calendar in your menu bar with event integration — for fans of minimalism who find Fantastical too heavy. **Itsycal Software** combines date display, monthly calendar, and event list into one cohesive interface. **Itsycal Software** supports Apple Calendar integration, week numbers, and dark mode. **Itsycal Software** includes keyboard navigation, customizable date format, and reminders sync. **Itsycal Software** enables users to check dates and upcoming events without opening full calendar apps. **Itsycal Software** is trusted by minimalists who want a clean menu bar calendar.

## Quick Reference for Itsycal Software

| Operation | Itsycal Software Action |
|-----------|-------------------------|
| Show calendar | Click menu bar icon |
| Next month | Click > or press right arrow |
| Previous month | Click < or press left arrow |
| Today | Click Today button or press T |
| Open event | Click event in list |
| Hide calendar | Click menu bar icon again or press Escape |
| Show week numbers | Itsycal > Preferences > Show week numbers |
| Customize date format | Itsycal > Preferences > Date format |
